SI-1452 Israeli Standard

Maps standard English key placements to the standard Hebrew keys (and vice versa) utilizing character frequency analysis.

✨

Hebrew Grammar Rules

Applies Hebrew final-form letters (ך, ם, ן, ף, ץ) intelligently based on word endpoints, preventing misplaced forms.

🚀

ZeroEntitlements Native API

Leverages low-level Carbon APIs for hotkey listeners, avoiding permissions requests until typing conversion is triggered.
